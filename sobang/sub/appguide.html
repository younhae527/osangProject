안전신고 등록 안내

  • 1안전신고/해양안전신고/안전제안 선택 시, 내용을 작성하는 단계로 이동합니다.
  • 1[개인정보 수집 및 이용안내] 안전신고/안전제안 선택 후 로그인 화면으로 이동합니다.
  • 2회원일 경우 아이디와 비밀번호를 입력하고 확인 버튼을 클릭합니다.
  • 3비회원일 경우 이름/휴대전화/이메일/주소를 입력 후 본인인증을 클릭, G-PIN이나 휴대전화 인증을 선택하여 인증합니다. 인증완료 후 확인을 누르면 로그인이 됩니다.
  • 1[개인정보 수집 및 이용안내] 개인정보 수집 동의에 체크 후 신청인 본인 정보를 입력합니다. 신청인 구분을 선택합니다.
  • 2[신고발생지역]신고발생 지역이 주소지와 동일한지 여부를 체크 후 아니면 주소 찾기를 눌러 위치를 찾습니다.
  • 3[진행방식 통보방식]셀렉트 박스 클릭 시 아래로 통보방식이 나열되며 클릭 시 선택됩니다.
  • 4통보방식은 이메일/SMS/서신/이메일+SMS/이메일+서신/SMS+서신/이메일+SMS+서신 입니다.
  • 1클릭 시 위치를 볼 수 있는 지도가 나타나며 팝업 창으로 GPS연동 동의/미동의 여부를 확인합니다.
  • 2지도를 클릭하면 선택한 위치에 신고 위치 심볼과 주소가 나타나며, 선택 위치를 신고위치로 사용합니다.
  • 1-1[안전신고/해양안전신고 작성] 제목과 내용을 입력합니다.
  • 1-2[안전제안 작성]제목과 개요/현황 및 문제점/개선방안/기대효과를 입력합니다
  • 1[사진/동영상/파일 추가]찾아보기.. 선택 후, 사진/동영상/파일을 추가 합니다.
  • 2[신고공유여부] 신고공유여부를 선택합니다.
  • 3이전 버튼을 선택 시 이전 화면으로, 다음 버튼 선택 시 기관선택 화면으로 이동합니다.
  • 1중앙행정기관을 선택합니다.
  • 2하위기관을 선택합니다.
  • 3처리기관을 잘 모르거나 여러 부처에 관련된 민원의 경우 처리기관 지정요청을 선택합니다.
  • 4미리보기 클릭 시 본인이 작성한 신고 및 제안을 볼 수 있습니다.
  • 5[신청] 버튼 클릭 시 안전신고/제안이 등록됩니다.
  • 1본인이 신청한 신고/제안을 [나의신고] 메뉴에서 확인할 수 있습니다.
  • 2최신 등록한 신고/제안이 목록의 최 상단에 위치하며 선택 시 상세내용을 조회 할 수 있습니다.
  • 3진행중인 신고/제안은 취하 버튼을 클릭하여 취하 할 수 있습니다.단, 이미 처리 완료된 신고/제안에 대해서 삭제 불가합니다.
